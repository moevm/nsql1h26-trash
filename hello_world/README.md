# 27 Сервис по вывозу мусора (Приложение одновременно для курьеров и клиентов)

## Web-приложение
- Главная страница с тремя видами пользователя: Админ, Курьер, Клиент
- Курьер и Клиент входят через обычные формы, для админа отдельная панель.

### Роли
**Курьер:**
- Будет панель, где будет список активных заказов, на которые он может откликнуться в зависимости от его местоположения.
Личная статистика по заказам(финансов)

**Клиент:**
- Будет форма для создания заказа: адрес, номер телефона, вид мусора, масса и объем, ФИО.
- История заказов
- Вкладка “Активные заказы”

**Админ:**
- Статистика по курьерам и заказам

## Используемые инструменты
- Бэкенд:ArangoDB + Python(FastApi), библиотека - python-arango, docker
- Фронтенд: React
## Структура кода
- каталог /api - REST API
- каталог /services - Бизнес логика
- каталог /db - логика общения с БД: Arango-запросы

## Инструкция по запуску
### 1. Клонирование репозитория
Откройте терминал и выполните команду для скачивания проекта:
```bash
git clone https://github.com/moevm/nsql1h26-trash.git
cd nsql1h26-trash/hello_world
```
### 2. Запуск через Docker Compose
Убедитесь, что у вас запущен Docker, и выполните:
```bash
docker compose up --build
```
### 3. Веб-интерфейс базы данных
После запуска база доступна по адресу: http://localhost:8529
